Game Data Aggregator

Описание проекта:

Game Data Aggregator - консольное Java-приложение для агрегации данных из нескольких открытых REST API игровой тематики.

Приложение выполняет опрос внешних источников по HTTP, получает данные в формате JSON, преобразует их в Java-объекты и сохраняет агрегированный результат в файл. Поддерживаются два формата хранения данных: JSON и CSV.

В рамках проекта приложение опрашивает 4 игровых API (например, Steam, CheapShark и др) после чего объединяет данные в едином формате для дальнейшего просмотра и анализа.

Проект разработан в рамках курсовой работы по дисциплине "Объектно-ориентированное программирование". Основные требования включают поддержку нескольких API, многопоточность (параллельный опрос источников), управление количеством одновременно выполняемых задач, настройку интервала между запросами, разделение бизнес-логики и пользовательского ввода, а также сохранение результатов в файл.


Текущий статус проекта:

На данный момент в проекте завершены 2 этапа из 3, предусмотренных методическими указаниями.

Реализованы:
- получение и агрегация данных из нескольких REST API;
- сохранение данных в файлы формата JSON и CSV;
- автоматический и интерактивный режимы работы;
- параллельный опрос источников;
- управление количеством одновременно выполняемых задач;
- периодический опрос API с заданным интервалом;
- потокобезопасная запись данных и корректное завершение работы.

В качестве следующего этапа осталось реализовать модульное тестирование с использованием JUnit и довести покрытие кода до требований курсовой работы.


Используемые технологии:
•	Java
•	Maven
•	java.util.concurrent
•	OkHttp - HTTP client для работы с REST API
•	Jackson - для парсинга JSON
•	Apache Commons CSV - для работы с CSV
